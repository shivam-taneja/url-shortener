# URL Shortener API

A backend REST API that shortens URLs and tracks click analytics.

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## Features

- Create a short URL from a long URL
- Redirect to the original URL using the short code
- Track how many times a short URL was clicked
- View analytics for a specific short URL
- Delete a short URL
